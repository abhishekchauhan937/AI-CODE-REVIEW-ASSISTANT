# 🤖 AI Code Review Assistant

A modern web-based AI Code Review Assistant that analyzes source code and provides intelligent feedback on code quality, readability, bugs, and performance. The application features an interactive dashboard, review history, charts, dark mode, authentication pages, and downloadable reports.

---

## 🚀 Features

- 🔍 AI-based Code Review
- 📊 Code Quality Statistics Dashboard
- 📈 Interactive Charts (Chart.js)
- 🌙 Dark Mode
- 📋 Copy Review to Clipboard
- 📄 Download Review Report
- 📝 Review History
- 🔎 Search Review History
- 🗑️ Clear Review History
- 🔐 Login & Signup Pages
- 💾 Local Storage Support
- 📱 Responsive User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## ▶️ How to Run

1. Clone the repository

```bash

```

2. Open the project folder.

3. Open `index.html` using Live Server or any modern web browser.

4. Start reviewing your code.
